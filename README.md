This project is a full-stack e-commerce web application in it's early development.

Backend:

- Java Spring Boot
- Spring Data JPA / Hibernate
- MySQL Database
- Swagger UI 

Frontend:

- React 
- CSS
- Canvas API

Features:

-  Functional Order placement with product selection list
-  Functional User registration form
-  Modern UI animated background
-  Animated popups
-  Order validation
-  Product catalog 
-  Order history
-  Responsive UI

Coming Soon:

- product Information
- Payment integration
- Customer reviews
- Chat support 
- Sign in


To get started:

1. Install MySQL and create database `eshop_db`
2. Run the SQL script in `/database/eshop_db.sql`
3. Update `application.properties` with your DB credentials
4. Run `EshopApiApplication.java`
5. API runs on `http://localhost:8080`
6. Swagger UI at `http://localhost:8080/swagger-ui/index.html`

Frontend:
1. `cd eshop-frontend`
2. `npm install`
3. `npm start`
4. App runs on `http://localhost:3000`
